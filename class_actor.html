Description détaillée

Base de tous les objets de la scène.

Cette classe est la classe de base de tous les acteurs, dont hérite toutes les classes pouvant être ajoutées a un monde. Elle définit les propriétés de bases d'un acteur, un constructeur de copie utilisant les méta-classes de Qt, ainsi que les méthodes virtuelles a implémenter par les classes enfants.
